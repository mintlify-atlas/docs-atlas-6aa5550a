## Lessons Overview

After selecting your grade, you'll access a collection of lessons designed to teach specific topics in Costa Rican social studies. Each lesson combines educational content with multimedia elements and interactive quizzes.

## Browsing Lessons

<Steps>
  <Step title="View Lessons List">
    After selecting a grade from the home page, you'll see the lessons list view displaying all available lessons for that grade level.
  </Step>
  
  <Step title="Review Lesson Information">
    Each lesson card shows:
    - Lesson title
    - Brief description of the topic
    - A representative image
    The lessons are displayed in a grid layout for easy browsing.
  </Step>
  
  <Step title="Select a Lesson">
    Click on any lesson card to open the full lesson view and start learning.
  </Step>
</Steps>

<Info>
  The page header displays your current grade level (e.g., "4to Grado") along with an encouraging message: "Explora las lecciones y pon a prueba tus conocimientos"
</Info>

## Lesson Structure

Each lesson follows a consistent structure:

### Header Section
- **Badge**: "Lección Interactiva" label in green
- **Title**: Large, prominent lesson title
- **Navigation**: Back link to return to the lessons list

### Content Sections
Lessons are organized into multiple sections, each containing:
- **Section Title**: Clear heading for each topic area
- **Text Content**: Educational content with HTML formatting for emphasis
- **Video Content** (optional): Embedded YouTube videos for visual learning

<Note>
  Content sections appear with a staggered animation effect, making it easier to focus on one section at a time as you scroll.
</Note>

## Studying Tips

<Tip>
  Use the **← Volver a Lecciones** link at the top to return to the lessons list at any time without losing your place.
</Tip>

### Reading Content
- Lesson content is displayed in a centered, readable format (max-width: 800px)
- Text is presented with ample spacing for comfortable reading
- HTML formatting highlights important terms and concepts

### Watching Videos
When a section includes video content:
- Videos are embedded directly in the lesson
- The player supports full screen viewing
- Videos complement the written content with visual explanations

## Navigation Features

### Breadcrumb Navigation
The lesson view provides clear navigation paths:
- **From Lesson View**: Click "← Volver a Lecciones" to return to the lessons list
- **From Lessons List**: Click "← Volver a Grados" to return to grade selection

### Invalid Lesson Handling
If you try to access a lesson that doesn't exist, the system automatically redirects you back to the lessons list for that grade.

## Progressing Through Content

<Steps>
  <Step title="Read Each Section">
    Work through the lesson sections in order. Each section builds on previous knowledge.
  </Step>
  
  <Step title="Watch Supplementary Videos">
    If a section includes a video, watch it to reinforce the concepts covered in the text.
  </Step>
  
  <Step title="Complete the Quiz">
    At the end of most lessons, you'll find an interactive quiz section marked with a flag emoji (🏁) and the heading "¡Pon a prueba lo aprendido!"
  </Step>
</Steps>

<Info>
  The quiz section appears after all lesson content with an encouraging message: "Responde las siguientes preguntas para completar la lección."
</Info>

## Special Grade: Resumen General

The "Resumen General" grade level provides comprehensive MEP 2026 exam preparation material. It follows the same lesson structure but covers a broader range of topics across all grade levels.

When viewing Resumen General content:
- Header shows: "Resumen General"
- Subtitle reads: "Material de preparación integral MEP 2026"
- Content is designed for comprehensive review
